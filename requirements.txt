# ====================
# INSTRUÇÕES DE INSTALAÇÃO
# ====================

# 1. Instalar Node.js (versão 18+)
#    Download: https://nodejs.org/

# 2. Instalar Expo CLI globalmente
#    npm install -g expo-cli

# 3. Na pasta do projeto, instalar dependências
#    npm install

# 4. Para adicionar Camera, Calendar e Location
#    npx expo install expo-camera expo-media-library expo-calendar expo-location react-native-maps

# 5. Para executar o app
#    npm start          (inicia o Expo Dev Client)
#    npm run android    (em emulador/dispositivo Android)
#    npm run ios        (em simulador/dispositivo iOS)
#    npm run web        (em navegador web)

# ====================
# NOTAS IMPORTANTES
# ====================

# - Certifique-se de ter um dispositivo ou emulador instalado antes de executar
# - Para iOS, você precisa do Xcode instalado
# - Para Android, você precisa do Android Studio ou SDK instalado
# - Algumas funcionalidades (câmera, localização, etc.) só funcionam em dispositivos reais ou emuladores/simuladores
# - Firebase requer configuração de credenciais via firebaseConfig.js
